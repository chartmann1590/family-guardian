Native iOS sideloading

Family Guardian can build an unsigned iOS IPA in GitHub Actions. You do not need a Mac locally. A stock iPhone still requires Apple signing at install time, so use a free Apple ID with Sideloadly or AltStore on Windows.

Free Apple ID limitation: free sideloaded apps usually need to be refreshed or reinstalled every 7 days. A paid Apple Developer Program account improves distribution, but is not required for this free path.

Build the IPA

  1. Open the repository on GitHub.
  2. Go to Actions -> iOS Unsigned IPA.
  3. Run the workflow manually, or let it run on changes under ios-app/.
  4. Download the FamilyGuardian-unsigned-ipa artifact.

Install from Windows

  1. Install Sideloadly or AltStore.
  2. Connect the iPhone over USB and trust the computer.
  3. Select FamilyGuardian-unsigned.ipa.
  4. Sign with your free Apple ID and install.
  5. On the iPhone, trust the developer profile if iOS asks.

Testing

  1. Run the Family Guardian server on your LAN with docker compose up --build.
  2. Open the iOS app and set the server URL to http://YOUR-LAN-IP:8080.
  3. Sign in or join with an invite code.
  4. Grant location permissions. Choose Always for background location testing.
  5. Verify the iPhone marker appears in the dashboard and Android app.

Known limits

  • No APNs remote push in v1. The app uses local notifications for WebSocket events while running.
  • Background GPS depends on iOS permission and power management.
  • The unsigned IPA is not installable directly; Sideloadly/AltStore signs it during install.
