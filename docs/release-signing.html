Release Signing

Android Signed APK

  1. Generate a keystore: keytool -genkey -v -keystore release.keystore -alias family-guardian -keyalg RSA -keysize 2048 -validity 10000
  2. Base64-encode it: base64 release.keystore > keystore.b64
  3. Add GitHub secrets: ANDROID_KEYSTORE_BASE64, ANDROID_KEYSTORE_PASSWORD, ANDROID_KEY_ALIAS, ANDROID_KEY_PASSWORD
  4. Push a tag — the android-release.yml workflow produces a signed APK

iOS Signed IPA

  1. Requires an Apple Developer Program membership
  2. Add GitHub secrets: certificate + provisioning profile via apple-actions/import-codesign-certs
  3. Push a tag — the ios-release.yml workflow produces a signed IPA
  4. Without Apple secrets, the workflow falls back to an unsigned IPA
