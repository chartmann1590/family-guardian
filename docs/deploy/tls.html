HTTPS / TLS Setup

Family Guardian can use Caddy as a TLS-terminating reverse proxy with automatic Let's Encrypt certificates.

Quick Start

  1. Copy docs/deploy/Caddyfile.example to docs/deploy/Caddyfile
  2. Edit the Caddyfile — replace your.domain with your actual domain
  3. Ensure DNS points to your server
  4. Open port 80 and 443 on your firewall
  5. Run: docker compose --profile tls up -d

Environment Variables

VariablePurpose
VAPID_PUBLIC_KEYWeb Push public key
VAPID_PRIVATE_KEYWeb Push private key
SESSION_SECRETCookie encryption secret (required in prod)

Generating VAPID Keys

npx web-push generate-vapid-keys
